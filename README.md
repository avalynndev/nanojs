# What is nanojs?

A framework made using react and vite. This is experimental.. doesn't really function that well.

Here is the [video](https://www.youtube.com/watch?v=uskQAvCheA0) showing how it works.

# How to run it yourself

### What you need

- Node.js 20 or newer
- npm or bun

### Setup

1. Download the code:
```bash
git clone https://github.com/avalynndev/nano-test.git
cd nano-test
```

2. Install stuff:
```bash
npm install
```

3. Start it up:
```bash
npm run dev
```

5. Go to [http://localhost:5173](http://localhost:5173)

### Make it live
```bash
vite run build
vite start
```
